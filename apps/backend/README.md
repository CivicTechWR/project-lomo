# LoMo Backend

The backend for LoMo — built with Django and PostGreSQL

## Stack

- **Django 5.0.4**
- **PostGreSQL 17**

## Dev

From the repo root:

```bash
docker compose build
docker compose up -d
```
Runs at `http://localhost:8000`.

## Commands

| Command               | Description              |
| --------------------- | ------------------------ |
| `docker compose stop` | Stops the containers     |
